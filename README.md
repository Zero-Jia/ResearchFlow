# JobKG-Agent

基于知识图谱 + LangChain v1 Agent
的智能职位推荐系统（支持多会话聊天与用户系统）

------------------------------------------------------------------------

## 1. 项目简介

JobKG-Agent 是一个面向计算机及人工智能专业学生的 AI 职业助手系统。

项目结合 **知识图谱（Neo4j）+ LangChain Agent + 多轮对话系统**，实现：

-   职位推荐
-   技能差距分析
-   学习路径规划
-   多轮上下文对话

系统支持用户登录后进入聊天界面（类似
ChatGPT），在不同会话中进行职业咨询。

------------------------------------------------------------------------

## 2. 核心功能

### 🤖 AI 职业推荐

-   职位推荐
-   技能差距分析
-   课程推荐
-   岗位对比
-   结构化输出

### 💬 多会话聊天系统

-   多 session 管理
-   上下文记忆
-   聊天记录存储
-   自定义会话名称

### 👤 用户系统

-   注册 / 登录（JWT）
-   鉴权机制

------------------------------------------------------------------------

## 3. 技术栈

### 后端

-   FastAPI
-   SQLAlchemy
-   MySQL
-   Neo4j
-   LangChain v1
-   LangGraph

### 前端

-   Vue3
-   Vite
-   Axios

------------------------------------------------------------------------

## 4. 启动方式

### 后端

cd backend\
source venv/bin/activate\
uvicorn app.main:app --reload

### 前端

cd frontend\
npm install\
npm run dev
